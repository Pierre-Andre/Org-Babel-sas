* Org-Babel-sas
Support for SAS language in org-babel

** Installation
add the following to your =.emacs= (I suppose that =ob-sas.el= was downloaded and pu in the 
directory =/home/foouser/.emacs.d/lisp/=)
#+BEGIN_SRC elisp
(load "/home/foouser/.emacs.d/lisp/ob-sas.el")
(require 'ob-sas)
#+END_SRC
